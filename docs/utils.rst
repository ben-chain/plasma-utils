=========
Utilities
=========

Description
===========
``plasma-utils.utils`` provides miscellaneous utilities.

API
===
.. code-block: javascript

    int32ToHex(x)

Converts a 32 byte integer to a hex string.

----------
Parameters
----------

1. ``x`` - ``Number``: A 32 byte integer.

-------
Returns
-------

``String``: The integer represented as a hex string.
